INSIGHT
- the largest numbers "bubble up" to the end of the array one at a time
- implemented an early-exit strategy using a 'swapped' flag to optimize for
    nearly sorted arrays

COMPLEXITY
- time:
    - average: Θ(n²)
    - best: Θ(n)
        - the array is already sorted
    - worst: Θ(n²)
        - the array is reverse sorted
- space: Θ(1) auxiliary
    - in-place (no auxiliary structures are built)
